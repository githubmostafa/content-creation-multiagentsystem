Content Creation Multi Agent AI System

A full stack AI powered content creation platform built using Google Agent Development Kit ADK Gemini 2.5 Flash FastAPI and React deployed on Google Cloud Run with Agent Runtime

Project Overview

This project is a multi agent AI system that transforms a single topic into a complete content package using a coordinated pipeline of specialized AI agents

Instead of a simple chatbot this system simulates a real world AI content studio where multiple agents collaborate to research draft improve and publish content

How It Works

User submits a topic
Orchestrator agent triggers all agents
Each agent performs its task research draft improve validate optimize
Final structured content is returned to the UI
User receives blog post SEO data social media posts email version
